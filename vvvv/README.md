# VL.Nodetool

**Status: Experimental**

A vvvv/VL integration for [NodeTool](https://nodetool.ai), enabling AI workflow execution within the vvvv visual programming environment.

For use with vvvv, the visual live-programming environment for .NET: http://vvvv.org

## Getting Started

1. Install via commandline as [described here](https://thegraybook.vvvv.org/reference/hde/managing-nugets.html):

   `nuget install VL.Nodetool -pre`

2. Open the help patches via the [Help Browser](https://thegraybook.vvvv.org/reference/hde/findinghelp.html)
