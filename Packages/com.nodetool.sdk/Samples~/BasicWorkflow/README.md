# BasicWorkflow Sample

1. Import the package into a Unity project.
2. Open the `BasicWorkflowExample` scene.
3. Add a `NodetoolSettings` asset under `Resources/NodetoolSettings` and configure endpoints.
4. Enter a prompt and press **Run** (or press Play to auto-run with the default prompt).

The script builds a minimal UI at runtime if you do not assign UI references in the Inspector.
